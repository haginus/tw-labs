Exercițiul 1

Creați un element cu ID-ul container care să conțină drept copii direcți alte 5 elemente cu clasa square. Scrieți cod CSS care să rezolve următoarele cerințe:

  • la lățimea ferestrei de peste 800px:
    • elementul cu ID-ul container să aibă lățimea și înălțimea de 500px, să fie centrat pe orizontală în pagină, să aibă un fundal verde și o bordură neagră de un pixel.
    • elementele cu clasa square de pe pozițiile impare să aibă bordură roșie, iar cele de pe pozițiile pare bordură albastră. Acestea vor fi poziționate pe cinci coloane egale, având înălțimea egală cu cea a părintelui și o bordură albastră.
  • la lățimea ferestrei de sub 800px:
    • elementul cu ID-ul container va avea lățimea egală cu jumătate din cea a paginii, iar elementele cu clasa square vor fi poziționate unul sub altul, având lățimea egală cu cea a părintelui.
    • culoarea fundalului paginii se va schimba lent, timp de o secundă de la alb la galben și repetându-se la infinit.

Exercițiul 2

Creați un document HTML și scrieți cod JavaScript care rezolvă următoarele:

  • la încărcarea paginii, se vor genera N paragrafe ce conțin prenumele vostru, unde N = max(10, K), unde K este numărul de minute al timpului curent.
  • Fiecare paragraf va primi aleatoriu o clasă din lista , clase pe care le veți defini ca setând fiecare o culoare diferită a textului.
  • La click oriunde pe fereastră în afara paragrafelor, se va afișa într-o fereatră de tip alert coordonata orizontală a click-ului.
  • La click pe oricare paragraf, vor fi șterse toate paragrafele ce au aceeași clasă cu acesta.

Exercițiul 3

Scrieți un document HTML care să conțină un input de tip text cu ID-ul date și un element de tip cu opțiunile "red", "green" și "blue", având selectată inițial prima opțiune. Scrieți cod JavaScript care să realizeze următoarele:

  • La încărcarea paginii, se va afișa în input data curentă, textul din input fiind colorat cu culoarea opțiunii selectate. O dată la 3 secunde, se va selecta următoarea opțiune din select, textul opținând noua culoare. După ultima opțiune se va selecta prima din listă.
  • La apăsarea tastei s, selecția automată se va opri.
  • La reîncărcarea paginii textul va râmâne colorat cu ultima culoare selectată înaintea reîncărcării.

Exercițiul 4

Scrieți un document HTML care să conțină un formular cu un element input de tip text și un buton de tip submit. Fie următorul array de obiecte stocat într-o variabilă pe serverul Node.js:

Scrieți codul necesar astfel încât la trimiterea formularului cu un anume nume, să se afișeze în pagină dacă persoana căutată este băiat sau fată și dacă este minor sau major. În cazul în care numele respectiv nu există în listă, se va afișa mesajul "Numele căutat nu există".

Exemplu: Pentru Gabriel, se va afișa băiat, nu este minor.
