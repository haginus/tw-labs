Ce este CSS?

Cascading Style Sheets (CSS) este limbajul pe care îl folosim pentru a stiliza paginile web. Acesta descrie cum sunt afișate elementele HTML pe ecranul utilizatorului.

Sintaxă

Pentru a stiliza elemente în CSS, folosim următoarea sintaxă: . Observați faptul că proprietățile CSS sunt despărțite între ele de ;, fiecare bloc de proprietăți CSS fiind inconjurat de acolade și aplicându-se elementelor HTML ce sunt găsite de selector.

Moduri de includere

Există trei moduri de a stiliza elementele HTML:

Inline styling

Stilizarea în linie ne permite să adăugăm stil direct către un element HTML folosind atributul style. Deoarece ne adresăm unui singur element HTML, nu mai este nevoie de selector, fiind intodus doar blocul de proprietăți în interiorul valorii atributului. Această metodă este puternic nerecomandată, deoarece crează redundanță (deci modificările ulterioare vor fi mai greu de făcut) și face codul mai puțin citibil.

Internal styling

Avem posibilitatea să adăugăm CSS direct în fișierele HTML, adăugând tag-uri în head-ul fișierului. Deși această metodă este mult mai bună decât stilizarea în linie, continuă să creeze redundanță din moment ce nu putem distribui codul între fișierele HTML.

External styling

Ne este permis să creem și să includem în HTML fișiere ce conțin doar cod CSS (stylesheets). Asfel putem avea unul sau mai multe fișiere de stiluri care sunt distribuite între mai multe fișiere HTML. De exemplu, acestea pot conține stiluri pentru elemente comune, cum ar fi header-ul paginii, navigația, diferite elemente custom, etc. Pentru a ne folosi de stilurile dintr-un fișier CSS, adăugăm în head-ul fișierului HTML următorul cod: .

Culori

Există mai multe moduri prin care se pot defini culori în CSS. Culorile create pot fi folosite oriunde, fie pentru modificarea culorii de fundal, a textului sau alte elemente.

Sistemul standard folosit este cel pe 24 de biți, ceea ce ne permite să generăm în jur de 16,7 milioane de culori, prin combinarea celor 3 canale de roșu, galben și albastru, fiecare putând avea o valoare între 0 și 255 (astfel obținem 2563 combinații posibile).

Valori hexadecimale RGB

Putem defini culori în CSS folosind coduri hexadecimale (hex codes). Fiecare cod hexadecimal este precedat de un diez (#) și apoi 6 cifre hexadecimale (în baza 16 - 0123456789abcdef). Aceste cifre se citesc în perechi de câte două, reprezentând canalele RGB (astfel avem 16 x 16 = 256, pentru fiecare canal).

Chiar dacă aceste coduri sunt mai greu de citit de către un om, sunt mult mai versatile decât metoda cu keyword-uri, permitându-ne să generăm tot spectrul de 16,7 milioane de culori.

Valori RGB și RGBA

Spre deosebire de varianta codurilor hexadecimale, această metodă ne permite să setăm culoarile rămânând în baza 10, definirea unei culor făcându-se astfel: rgb(r, g, b), unde r, g și b sunt valori întregi între 0 și 255, reprezentând cele 3 canale de culori.

Pentru a genera culori translucide, putem folosi funcția rgba(r, g, b, a), ce adaugă cananul alpha - un număr fracționar cuprins între 0 și 1 ce ne permite să setăm opacitatea culorii (0 fiind transparent și 1 total opac).

Proprietăți CSS

Proprietățile CSS au uneori sintaxe diferite în ceea ce privește setarea valorilor acestora. În special, proprietățile shorthand (care permit setarea mai multor proprietăți deodată) au o anumită sintaxă specifică. Totuși, există două valori pe care orice propietate CSS le acceptă:

  • initial - setează proprietatea CSS la valoarea ei inițială;
  • inherit - setează proprietatea CSS la valoarea moștenită de la părinte;

Deci, pentru următoarele proprietăți enumerate se va face abstracție că aceste două valori sunt posibile.

Display și vizibitate

display

Sintaxă:

  • block - afișează elementul ca un element block (precum ). Începe pe o linie nouă și ocupă toată linia.
  • inline - afișează elementul ca un element inline (precum ). Acesta ocupă doar spațiul de care are nevoie, iar proprietățile de dimenionare nu funcționează.
  • inline-block - afișează elementul ca un element inline-block. Deși este formatat în linie, acesta poate conține alte elemente de tip block și este permisă schimbarea dimensiunilor.
  • none - ascunde complet elementul. Este ca și cum acesta nu ar exista în pagină.

Notă: Există mai multe valori pentru display pe care le vom descoperi în laboratoarele viitoare.

overflow

Setează comportamentul scrollului în cazul în care dimensiunea conținutului depășește dimensiunea setată a elementului HTML. Acesta funcționează pe două axe: axa x (pe orizontală) și axa y (pe verticală).

Sintaxă:
  • visible - (implicit) conținutul nu este tăiat și poate continua în afara elementului.
  • hidden - conținutul este tăiat în cazul în care depășește înălțimea elementului, însă nu există suport pentru utilizator astfel încât să scrolleze conținutul. Acesta poate fi scrollat doar programatic prin JavaScript.
  • clip - asemănător hidden, dar blochează toate posibilitățile de scrolling, inclusiv prin JavaScript.
  • scroll - conținutul este tăiat și browserele arată tot timpul scrollbar-uri care permit utilizatorului să deruleze conținutul.
  • auto - browserele arată scrollbar-uri doar dacă lungimea scroll-ului depășește lungimea elementului (deci dacă scrollarea este posibilă).

overflow este un shorthand care ne permite să setăm subproprietățile overflow-x și overflow-y deodată. Acestea pot fi setate separat și au precedență față de overflow.

Selectori CSS

CSS ne permite să selectăm diferite elemente HTML folisindu-ne de selectori.

Selectori de bază

Sintaxă Descriere
* Selectorul universal
Selectează toate elementele.
Exemplu: * va selecta toate elementele din document.
nume-element Selectorul de tip
Selectează toate elementele cu un anume tag.
Exemplu: p va selecta toate elementele de tip paragraf.
.nume-clasă Selectorul de clasă
Selectează toate elementele ce au o anumită clasă.
Exemplu: .title va selecta toate elementele care conțin clasa "title".
#idElement Selectorul de ID
Selectează elementul cu ID-ul specificat.
Exemplu: #toc va selecta elementul cu ID-ul "toc".
[attr] Selectorul de atribut
Selectează elementele care au atributul specificat.
Exemplu: [autoplay] va selecta toate elementele care au atributul autoplay setat (la orice valoare).

Selectorul de atribut

CSS ne permite să selectăm elemente și după valoarea atributelor. Iată o listă cu cele mai importante variații selectorului de atribut:

  • [attr] Selectează elementele care au atributul attr setat.
  • [attr=value] Selectează elementele al căror attr este fix egal cu value.
  • [attr~=value] Selectează elementele al căror attr este o listă de cuvinte separate prin spațiu, unul dintre ele fiind exact value.
  • [attr^=value] Selectează elementele al căror attr este prefixat value.
  • [attr$=value] Selectează elementele al căror attr este sufixat value.
  • [attr*=value] Selectează elementele al căror attr conține value.

Combinatori

Putem combina mai mulți selectori pentru a obține selectori combinați. Iată cum:

Sintaxă Descriere
A B Combinatorul descendent
Combinatorul " " (spațiu) ne permite să selectăm toate elementele B care sunt descendenți ai lui A.
Exemplu: div span selecteză toate elementele span din interiorul unui div.
A > B Combinatorul copil
Combinatorul > ne permite să selectăm toate elementele B care sunt copii direcți ai lui A.
Exemplu: ul > li selecteză toate elementele li încuibărite direct într-un ul.
A ~ B Combinatorul frate general
Combinatorul ~ ne permite să selectăm toate elementele B care urmează după A și împart același părinte.
Exemplu: p ~ span selecteză toate elementele span care urmează după p, imediat sau nu.
A + B Combinatorul frate adiacent
Combinatorul + ne permite să selectăm toate elementele B care urmează direct după A și împart același părinte.
Exemplu: h2 + p selecteză primul p după h2.

:empty Selectează elementele goale (care nu au copii).
:first-child Selectează elementul care este primul printre frații săi.
:last-child Selectează elementul care este ultimul printre frații săi.
:nth-child(An+B) Selectează elementul al cărui număr este egal cu An+B. n este un număr natural care începe de la 0.
Exemplu: :nth-child(3n+2) va selecta din 3 în 3 elemente începând cu al doilea element.
:first-of-type Selectează elementul care este primul de același tip printre frații săi.
:last-of-type Selectează elementul care este ultimul de același tip printre frații săi.
:nth-of-type(An+B) Selectează elementul al cărui număr printre elementele de același tip este egal cu An+B. n este un număr natural care începe de la 0.
Exemplu: p:nth-of-type(3n+2) va selecta din 3 în 3 paragrafe începând cu al doilea paragraf.
:not(selector) Selectează elementele care nu sunt selectate de selector.
Exemplu: :not(p) va selecta toate elementele care nu sunt de tip paragraf.

Pseudo-elemente

CSS ne permite să stilizăm o parte specifică a unui element prin selectorii de pseudo-elemente. Iată cei mai folosiți selectori de acest tip:

Sintaxă Descriere
::before Crează un pseudo-element care este primul copil al elementului selectat.
::after Crează un pseudo-element care este ultimul copil al elementului selectat.
::first-letter Selectează prima literă de pe prima linie a unui element de tip block.
::first-line Selectează prima linie a unui element de tip block.

Compunere și grupare

Compunere

Selectorii CSS pot fi compuși (alăturați fără spațiu) pentru a genera selectori complecși. Compunerea se face în următoarea ordine:

  1. selector de tip
  2. selector de id
  3. selectori de clasă (pot fi mai mulți)
  4. selectori de atribute (pot fi mai mulți)
  5. selector de pseudo-clase (pot fi mai mulți)
  6. selector de pseudo-elemente (pot fi mai mulți)

Exemplu: va selecta elementul de tip div cu ID-ul audi ce care clasele vehicle și car și valoarea atributului tabindex egală cu 2.

Grupare

Selectorii CSS pot fi de asemena grupați, mai mulți selectori împărțind astfel același block de proprietăți. Pentru a grupa mai mulți selectori, îi despărțim prin virgulă.
Exemplu: - paragraful și primul div din fiecare părinte au coloare roșie.

Specificitatea selectorilor

Pentru a determina care selector are precedență față de alții, CSS implementează un sistem de ranking ce funcționează în modul următor:

  • Se pleacă de la valoarea 0 pentru fiecare selector.
  • Se adaugă:
    • 100 pentru fiecare selectare prin ID
    • 10 pentru fiecare selectare a unei clase, unui atribut sau unei pseudo-clase
    • 1 pentru fiecare selectare a unui element sau pseudo-element

De exemplu, span#iepure.alb::first-child are precedență 121 (1 element, un ID, o clasă și o pseudo-clasă).

Trebuie reținut că stilizarea în linie are precedență mai mare decât codul CSS definit în tag-ul sau într-un fișier extern.

De asemenea, în cazul în care specificitatea a doi selectori este egală, precedență va avea cel care se regăște mai târziu în stylesheet (este după ca ordine).

O modalitate prin care putem impune o anumită valoare a unei proprietăți este regula !important. Aceasta va suprascrie chiar și stilizarea în linie. Exemplu: va seta culoarea pentru toate paragrafele ca fiind roșie.
